<strong>Free Models Router</strong>는 전 세계 사용자를 위한 DGrid의 무료 지능형 추론 게이트웨이입니다. 특정 무료 모델을 직접 선택하는 대신 안정적인 모델 ID 하나를 호출하면, DGrid가 서비스 가용성, 응답 효율, 작업 요구 사항을 기준으로 사용 가능한 모델 리소스에 요청을 자동 라우팅합니다.

<Info>
DGrid가 사용 가능한 무료 모델을 자동으로 선택하게 하려면 모델 ID `dgridai/free`를 사용하세요. 실시간 모델 정보는 [모델 상세 페이지](https://dgrid.ai/models/dgridai/free)를 참고하세요.
</Info>

## 적합한 사용 사례

Free Models Router는 특정 공급자 모델을 지정할 필요 없이 빠른 접근성과 낮은 진입 장벽이 중요한 일반 AI 사용에 적합합니다.

다음 작업에 잘 맞습니다.

- 일상적인 Q&A와 리서치 보조
- 다국어 작성, 재작성, 번역
- 코딩 지원, 디버깅 제안, 코드 설명
- 논리 분석과 구조화된 추론
- 학습 지원과 개념 설명
- 초안 작성, 요약, 콘텐츠 제작
- 범용 AI 역량이 필요한 복잡한 문제 해결

## 라우팅 방식

DGrid는 단일 라우팅 계층 뒤에 여러 무료 추론 용량을 집계합니다. 각 요청마다 라우터는 사용 가능한 모델 리소스를 평가하고 적합한 백엔드로 요청을 스케줄링합니다.

라우팅은 다음 요소를 고려할 수 있습니다.

- 현재 서비스 가용성
- 예상 응답 속도
- 작업 유형과 프롬프트 요구 사항
- 연결된 무료 모델 소스의 사용 가능한 컴퓨트
- 기본 라우트의 운영 안정성

따라서 기본 모델은 요청마다 달라질 수 있습니다. 애플리케이션에 고정된 모델 ID가 필요하다면 `dgridai/free` 대신 특정 Model API 모델 ID를 사용하세요.

## 빠른 시작

Free Models Router는 DGrid AI Gateway의 다른 모델과 동일한 OpenAI 호환 chat completion 인터페이스를 사용합니다.

```bash
curl https://api.dgrid.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $DGRID_API_KEY" \
  -d '{
    "model": "dgridai/free",
    "messages": [
      {
        "role": "user",
        "content": "Explain zero-knowledge proofs in simple terms."
      }
    ]
  }'
```

OpenAI SDK를 사용할 때는 DGrid base URL만 설정하면 됩니다.

```typescript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://api.dgrid.ai/v1',
  apiKey: process.env.DGRID_API_KEY,
});

const completion = await client.chat.completions.create({
  model: 'dgridai/free',
  messages: [
    {
      role: 'user',
      content: 'Help me outline a launch plan for a developer tool.',
    },
  ],
});

console.log(completion.choices[0].message.content);
```

## 사용 제한

무료 접근의 안정성과 공정성을 유지하기 위해 DGrid는 Free Models Router에 요청 제한을 적용합니다. 제한은 계정 단위로 적용됩니다.

| 접근 수준 | 자격 | 분당 요청 수 | 일일 요청 수 |
| --- | --- | ---: | ---: |
| 표준 접근 | 최소 `$5`의 적격 충전을 완료하지 않은 계정 | 10 | 100 |
| 상향 접근 | 최소 `$5`의 적격 충전을 완료한 계정 | 20 | 1,000 |

적격 충전은 계정 잔액에 직접 추가된 자금을 의미하며, 계정이 `$5` 기준을 충족했는지 판단하는 데 사용됩니다.

### 속도 제한 오류

적용 가능한 제한을 초과한 요청은 다음 메시지와 함께 거부됩니다.

```text
Rate limit exceeded. Please try again later.
```

애플리케이션에 예측 가능한 더 높은 처리량이 필요하다면 유료 모델 라우트를 사용하거나 DGrid 지원팀에 프로덕션 운영 지침을 문의하세요.

## 운영 권장 사항

- `dgridai/free`를 고정 모델이 아니라 편의 라우터로 취급하세요.
- 라우팅된 기본 모델의 공급자별 동작에 의존하지 마세요.
- 일시적인 속도 제한 또는 가용성 응답에 대해 백오프 기반 재시도 로직을 추가하세요.
- 결정적인 모델 선택, 더 높은 처리량, 더 강한 SLA 기대치, 모델별 기능이 필요하다면 유료 모델 ID를 사용하세요.
- 나중에 사용량을 대조해야 한다면 응답의 `DGrid-Request-ID`를 기록하세요. 요청 단위 결제 조회는 [요청 결제 세부 정보 조회](/ko/api-reference/usage-and-billing/get-request-billing-details)를 참고하세요.
