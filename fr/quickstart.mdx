## Commencer avec DGrid AI Gateway

DGrid AI Gateway vous offre une API unique, compatible OpenAI, pour plus de 200 modèles de premier plan. Cette section vous accompagne de zéro jusqu'à votre première complétion de chat.

<Steps>

<Step title="Obtenez votre clé API">

Obtenez une `DGRID_API_KEY` valide en suivant le [guide de création de clé](https://blog.dgrid.ai/posts/2026-01-04/).

<Note>
Le secret de votre clé n'est affiché qu'une seule fois, au moment de sa création. Copiez-le immédiatement et conservez-le dans un emplacement sécurisé, tel qu'un gestionnaire de secrets. Ne l'exposez jamais dans du code côté client ou dans des dépôts publics.
</Note>

</Step>

<Step title="Effectuez votre première requête">

Envoyez une requête de complétion de chat au point de terminaison de DGrid AI Gateway. Le paramètre `model` utilise le format `provider/model-name` (par exemple `openai/gpt-4o`), cohérent pour tous les modèles pris en charge par DGrid.

<CodeGroup>

```bash cURL
curl https://api.dgrid.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $DGRID_API_KEY" \
  -d '{
    "model": "openai/gpt-4o",
    "messages": [
      {
        "role": "user",
        "content": "What is the meaning of life?"
      }
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.dgrid.ai/v1",
    api_key="<DGRID_API_KEY>",
)

completion = client.chat.completions.create(
    model="openai/gpt-4o",
    messages=[
        {
            "role": "user",
            "content": "What is the meaning of life?"
        }
    ]
)

print(completion.choices[0].message.content)
```

```typescript TypeScript
import OpenAI from 'openai';

const openai = new OpenAI({
  baseURL: 'https://api.dgrid.ai/v1',
  apiKey: '<DGRID_API_KEY>',
});

const completion = await openai.chat.completions.create({
  model: 'openai/gpt-4o',
  messages: [
    {
      role: 'user',
      content: 'What is the meaning of life?',
    },
  ],
});

console.log(completion.choices[0].message.content);
```

</CodeGroup>

<Tip>
DGrid est entièrement compatible avec le SDK OpenAI — pointez `baseURL` vers `https://api.dgrid.ai/v1` et votre code existant fonctionne sans modification. Installez-le avec `pip install openai` ou `npm install openai`.
</Tip>

</Step>
