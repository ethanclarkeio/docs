Cette page rassemble des tutoriels d'intégration pratiques pour utiliser **DGrid AI Gateway** avec les outils et SDK courants des développeurs.

## Cursor (remplacer l'URL de base OpenAI)

1. Obtenez votre `DGRID_API_KEY` (voir le prérequis dans [AI Gateway](/fr/ai-gateway/overview)).
2. Ouvrez **Cursor Settings** → **Models**.
3. Sous **API Keys**, activez **Override OpenAI Base URL**.
4. Définissez l'URL de base sur :
   - `https://api.dgrid.ai/v1`
5. Collez votre `DGRID_API_KEY` dans **OpenAI API Key**.
6. Choisissez un nom de modèle pris en charge en utilisant le format DGrid (par exemple : `openai/gpt-4o`).

## OpenClaw

Vous pouvez configurer OpenClaw pour acheminer les requêtes via **DGrid AI Gateway** en utilisant le point de terminaison compatible OpenAI.

1. Ouvrez les paramètres de modèle/fournisseur d'OpenClaw.
2. Définissez l'URL de base de l'API sur `https://api.dgrid.ai/v1`.
3. Collez votre `DGRID_API_KEY` dans le champ de clé API.
4. Sélectionnez un identifiant de modèle DGrid pris en charge.
5. Enregistrez la configuration et effectuez une requête de test rapide.
