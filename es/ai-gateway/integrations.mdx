Esta página recopila tutoriales prácticos de integración para usar **DGrid AI Gateway** en herramientas y SDK de desarrollo comunes.

## Cursor (sobrescribir la URL base de OpenAI)

1. Obtenga su `DGRID_API_KEY` (consulte el requisito previo en [AI Gateway](/es/ai-gateway/overview)).
2. Abra **Cursor Settings** → **Models**.
3. En **API Keys**, habilite **Override OpenAI Base URL**.
4. Configure la URL base como:
   - `https://api.dgrid.ai/v1`
5. Pegue su `DGRID_API_KEY` en **OpenAI API Key**.
6. Elija cualquier nombre de modelo compatible utilizando el formato de DGrid (por ejemplo: `openai/gpt-4o`).

## OpenClaw

Puede configurar OpenClaw para enrutar las solicitudes a través de **DGrid AI Gateway** utilizando el endpoint compatible con OpenAI.

1. Abra la configuración de modelo/proveedor de OpenClaw.
2. Configure la URL base de la API como `https://api.dgrid.ai/v1`.
3. Pegue su `DGRID_API_KEY` en el campo de clave de API.
4. Seleccione un ID de modelo de DGrid compatible.
5. Guarde la configuración y ejecute una solicitud de prueba rápida.
