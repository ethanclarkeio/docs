## Comience con DGrid AI Gateway

DGrid AI Gateway le ofrece una API única, compatible con OpenAI, para más de 200 modelos líderes. Esta sección lo lleva desde cero hasta su primera finalización de chat.

<Steps>

<Step title="Obtenga su clave de API">

Obtenga una `DGRID_API_KEY` válida siguiendo la [guía de creación de claves](https://blog.dgrid.ai/posts/2026-01-04/).

<Note>
El secreto de su clave se muestra solo una vez en el momento de la creación. Cópielo de inmediato y guárdelo en un lugar seguro, como un gestor de secretos. Nunca lo expone en código del lado del cliente ni en repositorios públicos.
</Note>

</Step>

<Step title="Realice su primera solicitud">

Envíe una solicitud de finalización de chat al endpoint de DGrid AI Gateway. El parámetro `model` utiliza el formato `provider/model-name` (por ejemplo, `openai/gpt-4o`), consistente en todos los modelos compatibles con DGrid.

<CodeGroup>

```bash cURL
curl https://api.dgrid.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $DGRID_API_KEY" \
  -d '{
    "model": "openai/gpt-4o",
    "messages": [
      {
        "role": "user",
        "content": "What is the meaning of life?"
      }
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.dgrid.ai/v1",
    api_key="<DGRID_API_KEY>",
)

completion = client.chat.completions.create(
    model="openai/gpt-4o",
    messages=[
        {
            "role": "user",
            "content": "What is the meaning of life?"
        }
    ]
)

print(completion.choices[0].message.content)
```

```typescript TypeScript
import OpenAI from 'openai';

const openai = new OpenAI({
  baseURL: 'https://api.dgrid.ai/v1',
  apiKey: '<DGRID_API_KEY>',
});

const completion = await openai.chat.completions.create({
  model: 'openai/gpt-4o',
  messages: [
    {
      role: 'user',
      content: 'What is the meaning of life?',
    },
  ],
});

console.log(completion.choices[0].message.content);
```

</CodeGroup>

<Tip>
DGrid es totalmente compatible con el SDK de OpenAI: apunte `baseURL` a `https://api.dgrid.ai/v1` y su código existente funcionará sin cambios. Instálelo con `pip install openai` o `npm install openai`.
</Tip>

</Step>
