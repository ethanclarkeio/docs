本頁收錄了在常見開發者工具與 SDK 中使用 **DGrid AI Gateway** 的實用整合教學。

## Cursor（覆寫 OpenAI Base URL）

1. 取得您的 `DGRID_API_KEY`（請參閱 [AI Gateway](/zh-Hant/ai-gateway/overview) 中的前置需求）。
2. 開啟 **Cursor Settings** → **Models**。
3. 在 **API Keys** 中，啟用 **Override OpenAI Base URL**。
4. 將 base URL 設定為：
   - `https://api.dgrid.ai/v1`
5. 將您的 `DGRID_API_KEY` 貼到 **OpenAI API Key**。
6. 使用 DGrid 格式選擇任何支援的模型名稱（例如：`openai/gpt-4o`）。

## OpenClaw

您可以設定 OpenClaw，透過相容 OpenAI 的端點，將請求路由至 **DGrid AI Gateway**。

1. 開啟您的 OpenClaw 模型／供應商設定。
2. 將 API base URL 設定為 `https://api.dgrid.ai/v1`。
3. 將您的 `DGRID_API_KEY` 貼到 API 金鑰欄位。
4. 選擇支援的 DGrid 模型 ID。
5. 儲存設定並執行一次快速測試請求。
