## 概覽

本服務條款（「條款」）規範對 DGrid AI Gateway（「AI Gateway」）的存取與使用，包括 DGrid 提供的相關 API、文件、工具、介面及服務功能。

AI Gateway 是一項用於存取、路由、管理與整合 AI 模型能力及相關功能的服務。除非 DGrid 以書面方式明確聲明，本條款專門適用於 AI Gateway 及相關服務功能。

## 本條款的適用範圍

本條款適用於您作為服務對 AI Gateway 的存取與使用。

為求明確，除非 DGrid 以書面方式明確聲明，本條款不適用於 DGrid Premium 或任何與 DGrid Premium 授權相關的產品。針對其他產品，DGrid 可能發布並適用獨立的產品特定或服務特定條款、政策、聲明或商業條款。

存取或使用 AI Gateway，即表示您同意遵守本條款，以及 DGrid 可能不時為 AI Gateway 發布的任何其他文件、政策、使用要求或技術指引。

## 資格與合規

僅在您依適用法律獲得授權的情況下，您方可使用 AI Gateway，並（如適用）代表您自己或您所代表的法律實體使用。

您有責任確保您對 AI Gateway 的存取與使用，符合與您的司法管轄區、產業及使用情境相關的所有適用法律、法規、制裁限制、出口管制要求及內部合規義務。

若存取或使用會造成法律、法規、安全或合規風險，DGrid 可能限制、暫停或拒絕對 AI Gateway 的存取。

## 帳戶與 API 憑證

要存取某些 AI Gateway 功能，您可能需要建立帳戶、取得 API 憑證，或使用 DGrid 發出的憑證。

您負責：

- 維護您帳戶與 API 憑證的機密性與安全性；
- 在您的帳戶或憑證下進行的所有活動；
- 確保憑證不會被分享、洩露、轉售或以未經授權的方式使用；以及
- 在發現任何可疑的未經授權存取、憑證洩露或濫用時，立即通知 DGrid。

DGrid 可能基於安全、運營、法律或合規目的，在合理必要時輪換、撤銷、暫停或更換憑證。

## 允許的使用

在符合本條款的前提下，您僅可將 AI Gateway 用於合法、經授權且有文件記載的目的。

您可依照 DGrid 發布的任何技術文件、使用限制、產品指引及服務特定限制使用 AI Gateway。

## 禁止的使用

您不得將 AI Gateway 用於：

- 違反任何適用法律或法規；
- 侵犯、不當使用或以其他方式違反 DGrid 或任何第三方的權利；
- 提交、傳輸或處理非法、有害、欺騙性、濫用性、侵權或其他被禁止的內容；
- 嘗試未經授權存取系統、帳戶、模型、資料或網路；
- 干擾、破壞、降低或規避 AI Gateway 或相關系統的安全性、完整性或可用性；
- 對該服務進行反向工程、提取、爬取、基準測試或複製，除非適用法律明確允許；
- 規避速率限制、存取控制、安全防護、地理限制或使用限制；
- 以可能對 DGrid、其供應商或其他使用者造成不合理的安全、法律、信譽或運營風險的方式使用 AI Gateway；
- 在未經 DGrid 書面明確許可的情況下，轉售、再授權或向未經授權的第三方提供對 AI Gateway 的存取；或
- 將 AI Gateway 用於 DGrid 文件、政策頁面或適用服務聲明中所列的任何禁止或受限使用情境。

DGrid 可能不時更新或補充適用於 AI Gateway 的禁止使用限制。

您有責任評估輸出內容的準確性、適用性、合法性以及是否適合您的預期用途。輸出內容可能不完整、不準確、存在偏誤、無法取得、延遲或不適用於某些使用情境，在未經適當審查與判斷前不應依賴這些內容。
