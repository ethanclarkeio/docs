## 概覽

本頁說明 DGrid AI Gateway（「AI Gateway」）的地區可用性相關注意事項。

AI Gateway 可能無法在所有國家、地區或司法管轄區內使用。可用性可能因法律、法規、制裁、出口管制、基礎設施、支付、運營或第三方供應商的限制而有所不同。

本頁專門適用於 AI Gateway。它不會取代適用於其他 DGrid 產品的任何獨立地區限制、法律聲明或產品特定條款。如需了解服務使用要求，請參閱[服務條款](/zh-Hant/ai-gateway/terms-of-service)。

## 受限或不支援的地區

DGrid 可能會在其認定存取或使用會造成法律、法規、制裁、出口管制、安全、運營或商業風險的任何國家、地區或司法管轄區，限制或拒絕對 AI Gateway 的存取。

在某些情況下，DGrid 可能會：

- 封鎖帳戶建立或 API 存取；
- 限制特定模型或功能的使用；
- 要求額外的驗證或審查；或
- 在無法允許或無法繼續提供服務時，暫停或終止存取。

使用者不得以規避地理、技術、合規或合約控制的方式，從受限或不支援的地區存取或使用 AI Gateway。

## 各地區的服務限制

即使在 AI Gateway 可用的地區，也未必所有功能、模型、端點、整合或帳戶選項都能在每個地區提供。

地區差異可能源於：

- 第三方模型或供應商限制；
- 當地合規要求；
- 資料路由或基礎設施限制；
- 支付或計費可用性；
- 語言或支援覆蓋範圍；或
- 運營推出時程。

DGrid 保留隨時變更、限制或撤銷任何 AI Gateway 功能在特定地區可用性的權利。

## 合規與出口管制

使用 AI Gateway 須遵守所有適用的貿易限制、制裁法規、出口管制規則及相關法規要求。

您有責任確保您對 AI Gateway 的存取與使用，符合對您、您的組織、您的使用者及您的使用情境所適用的所有法律與限制。

DGrid 可能要求在特定地區或特定使用情境下，提供額外的審查、文件、驗證或合約保證，才能啟用或持續提供存取。

## 地區可用性的變更

DGrid 可能隨時更新、擴展、限制、暫停或終止 AI Gateway 在任何地區的可用性，無論是否事先通知，惟須符合適用法律。

變更可能源於：

- 法律或法規的變動；
- 制裁或出口管制的發展；
- 第三方供應商支援的變動；
- 基礎設施、安全或運營問題；或
- DGrid 內部的合規或風險評估。

使用者應參閱最新發布的 DGrid 文件，以取得目前的地區可用性指引。
